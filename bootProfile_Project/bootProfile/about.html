bootProfile

bootProfile is a web based service that lets the user compare two or more stats of either Teams or Players against each other. It enabled you keep up to date with the latest happening in and around the world of football, Your favorite League, team and players. bootProfile keeps you up to date with team statistics, achievement, history and many more so also for Players as well.

It also allows the user to test if the data entered is correct or not by letting the user letting the users communicate with the administrators telling which data entered is wrong or not.
